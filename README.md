# 기능
1. 메인공간 입장
2. WASD 또는 화살표 키를 사용해 2D 캐릭터가 맵을 자유롭게 이동
3. NPC와 상호작용하여 미니게임 플레이 기능
4. 발판을 밟는식으로 상호작용을하여 텍스트 메세지 출력
5. 미니 게임 점수를 기록하고 UI에 실시간 표시
6. 미니 게임 종료 시 맵으로 복귀 및 결과 표시
7. 캐릭터가 이동할 때 카메라가 자연스럽게 따라오는 기능 구현.
